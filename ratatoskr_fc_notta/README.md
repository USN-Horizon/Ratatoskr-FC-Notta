## File Structure Format
```
ratatoskr_fc_notta/
|- ratatoskr_fc_notta.ino    <- main arduino .ino file
|- src/                      <- header/implementation files
|  |- task.h
|  |- task.cpp
|     ...
|- lib/                      <- project libraries
|  |- library1/...
|  |- library2/...
|     ...
|- include/                  <- project configuration / other useful headers
   |- pins.h
   |- config.h
      ...
```
